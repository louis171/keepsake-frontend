# Keepsake

Run:

### `npm install`
### `npm start`

## This project is the React frontend of [Keepsake-API](https://github.com/louis171/keepsake-api)

## Current status

* Pages
  - Home
  - About
  - Login
  - Register
  - My Profile
  - Logout

Able to create user account, sign in and create keepsakes and memories with images
